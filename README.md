# Anton Wester portfolio

Astro-based portfolio site for Anton Wester AB with localized landing pages for Gothenburg and nearby cities.

## Routes

English routes:

- `/` for Gothenburg
- `/stenungsund/`
- `/ale/`
- `/kungalv/`
- `/partille/`

Swedish routes:

- `/sv/` for Göteborg
- `/sv/stenungsund/`
- `/sv/ale/`
- `/sv/kungalv/`
- `/sv/partille/`

## Commands

- `npm run dev` starts the Astro dev server.
- `npm run build` creates the production build in `dist/`.
- `npm run preview` previews the production build locally.
